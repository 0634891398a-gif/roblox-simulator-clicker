## [v2.4.1] - 2023-10-15
### Changed
- Optimized click frequency settings for smoother performance in **Roblox Studio**.
- Updated the UI to provide better accessibility options for visually impaired users.

- Resolved an issue where the auto clicker would not activate in certain game modes, such as **Obby** and **Tycoon**.
- Patched a bug that caused crashes when switching between games with different input settings.

- Introduced a new feature to save and load click profiles, allowing users to optimize settings for different games like **Simulator** and **Adventure**.
- Added sound effects for click activation to provide feedback during gameplay.

### Changed
- Enhanced the performance of the clicker to reduce CPU usage during prolonged gaming sessions.
- Improved the compatibility with **Roblox's Anti-Cheat** system for safer usage.

### Fixed
- Fixed a bug that prevented users from using the clicker after minimizing the app during gameplay.
- Addressed a visual glitch that distorted the interface when running in high-DPI settings.
